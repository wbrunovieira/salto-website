V1 · Barras com largura das letras S · A · L — ascendente da esquerda

Cada barra cobre exatamente uma letra (S, A, L). Alturas ascendentes. A transparência mantém o texto legível através das barras.

V2 · Barras centradas no L — o L é o pico, barras flanqueiam

A barra mais alta está exatamente sobre o L. Duas barras menores flanqueiam (sobre A e T). O L é o pico visual da sequência.

V3 · Uma barra larga por trás — todo o wordmark como barra de fundo

Uma única barra de fundo (a mais alta) cresce por trás do wordmark completo. Mais clean — o texto está sobre a barra, não vice-versa.

V4 · Barras como sufixo — imediatamente após o wordmark

Wordmark intacto + barras como elemento sufixo. Muito coesas, sem quebrar a leitura. O símbolo "acompanha" a palavra.
